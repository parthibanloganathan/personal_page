E6B+ is an all-in-one solution to a pilot's preflight needs. It's a flight computer which performs several extra features like graphing weight and balance and displaying ground speed. To date, it has over 40,000 downloads on Google Play with a 4-star rating.

Hermes Keyboard

Contextually aware Slack extensions at the keyboard level. Having a conversation with a friend on a messaging app? Just type \uber to send them an Uber to your location or \spotify Radioactive to send them a song. Start chatting about restaurants and we'll pop Yelp results for you. Won First place at Greylock Hackfest 2015.

Check out the demo

Node Computing Platform

The Node Computing Platform solves large systems of diagonally dominant linear equations using an implementation of the Gauss-Seidel method on a network of nodes. It can solve a matrix consisting of hundreds and even thousands of variables using network based feedback to converge onto the solution. The web application was implemented using Java, PostgreSQL and frontend web technologies.

Solve hundreds of linear equations now

Sailthru Android SDK

The Sailthru Android library collects data on client app usage to create a personalized experience tailored for each user. The library efficiently stores data and sends it to back-end servers for analytics in the background. I also developed an Android application to demonstrate and test the functionality of the library.

View the test app code

Clapper

The Clapper circuit was built for my Intro to Electrical Engineering class. Inspired by the 1980's product, the clapper can switch on and an LED based on a clap sequence. I tuned it to activate the switch on three claps with at least one second betwen them. Most of this project was done with hardware. You can see the code and project report on GitHub.

Watch the demo

Living Language

Living Language won 1st place at Facebook's NYC hackathon. The chrome extension teaches you a new language by dynamically replacing words, sentences or paragraphs while you browse the web. It supports English translation and native pronunciations too. See Facebook's coverage on the hackathon at Facebook Engineering NYC. Coming to the Chrome Web Store soon.
