Node Computing Platform

The Node Computing Platform solves large systems of diagonally dominant linear equations using an implementation of the Gauss-Seidel method on a network of nodes. It can solve a matrix consisting of hundreds and even thousands of variables using network based feedback to converge onto the solution. The web application was implemented using Java, PostgreSQL and frontend web technologies.

Subway Info

Subway Info is a widget for the Dashing platform. This widget displays train arrivals in New York by scraping GTFS information from the MTA. It periodically anaylzes it for information on arriving trains at subway stops near you.

Sailthru Android SDK

The Sailthru Android library collects data on client app usage to create a personalized experience tailored for each user. The library efficiently stores data and sends it to back-end servers for analytics in the background. I also developed an Android application to demonstrate and test all the functionality of the library.
